CLI Installation Guide

Joys Fonts provides an easy-to-use CLI to download optimized font files and configure Tailwind CSS automatically without adding bundle size to your project.

⚡ Quick Start

Run the command below in the root of your project using your preferred package manager:

pnpm dlx joys-fonts add joys-vazir
npx joys-fonts add joys-vazir
yarn dlx joys-fonts add joys-vazir
bunx joys-fonts add joys-vazir

CLI Commands

  • add <slug>: Downloads assets and configures the font in your project.
  • list, ls: Lists all available fonts in the catalog.
  • help, -h: Displays the help screen.

Configuration Options

  • -d, --dest <path>: Customize the directory to store font assets.
  • -c, --css <path>: Path of the stylesheet to inject configuration.
  • -y, --yes: Runs the installer non-interactively using defaults.
